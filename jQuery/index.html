Résumé 1 :

Les sélecteurs :

  • Par la balise, exemple $("h1")
  • Par l'id, exemple $("#moi")
  • Par la classe, exemple $(".ma-classe")
  • Par lui meme, $(this)
  • Par le type, exemple $("input:text")
  • TOUS les types, exemple $(":text")
  • Par l'attribut, $("img[alt]")
  • TOUS les attributs, $("[alt]")
  • Par le nom d'attribut ET sa valeur, $("img[title='tigre']")
  • Selectionner PLUSIEURS sélecteurs, $("h1, div, p, div.ma-classe")

Les évenements Jquery :

  • click()
  • dblclick()
  • mouseenter()
  • mouseleave()
  • hover()
  • mousedown()
  • mouseup()
  • focus()
  • blur()

Les fonctions hide(), show() et toggle :

  • hide() permet de cacher ce qu'on a cible
  • show() affiche ce qu'on a cible
  • toggle() cache si affiché, affiche si caché. Intérrupteur qui fait les deux
  • "slow" fait l'action lentement
  • "fast" rapidement
  • 1000 fait l'action en 1000 millisecondes
